# Gesture-Controlled Drawing & 3D Rotation

A Computer Vision project built using Python, OpenCV, and MediaPipe that allows users to interact with the application using hand gestures.

## Features

- ☝️ One Finger → Draw on the canvas
- ✌️ Two Fingers → Switch to 3D Rotation Mode
- 🤟 Three Fingers → Clear the canvas

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run

1. Clone this repository
2. Install the required libraries:

```bash
pip install opencv-python mediapipe numpy
```

3. Run:

```bash
python hand_draw.py
```
